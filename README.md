# 🌍 MakeMemory — Airbnb Style Listing App
 A full-stack web application inspired by Airbnb where users can explore, create, edit, and delete travel listings.

---

##  Features

*  Add new listings (hotel, villa, resort, etc.)
*  Edit existing listings
*  Delete listings
*  View detailed listing pages
*  Image-based listings
*  Responsive UI using Bootstrap
*  Clean layout with navbar & footer

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, Bootstrap, EJS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB, Mongoose
* **Templating Engine:** EJS + EJS-Mate

## ⚙️ Installation & Setup

1️⃣ Clone the repository

```
git clone https://github.com/sunny-kumar06/MakeMemory.git
```

2️⃣ Install dependencies

```
npm install
```

3️⃣ Run the server

```
node app.js
```

4️⃣ Open in browser

```
http://localhost:8080
```
